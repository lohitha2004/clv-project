# Customer Lifetime Value (CLV) Prediction

## Overview
A Machine Learning project that predicts Customer Lifetime Value 
to help businesses identify high-value customers and drive revenue.

## Tech Stack
- Python
- Scikit-learn
- Machine Learning
- SQL
- Google Colab

## What it does
- Predicts future revenue value of each customer
- Identifies key customer segments driving business growth
- Performs data analysis and feature engineering

## How to Run
1. Open the `.ipynb` file in Google Colab
2. Run all cells sequentially
3. View predictions and customer segments
